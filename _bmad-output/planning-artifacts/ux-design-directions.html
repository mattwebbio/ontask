Direction Selection
"After exploring, which direction resonates — or which elements do you want to combine?"

Layout Intuition

  • Oracle's centered single-task — does the radical focus feel right?
  • Architecture's card structure — familiarity over novelty?
  • Editorial's type-led structure — appeals to design-literate?
  • Spatial's timeline — does your user think in time?

Visual Weight

  • Deep Clay's dark immersion — does the app feel heavy enough?
  • Oracle/Editorial's restraint — does terracotta only appear at meaning?
  • Signal's data density — does your user want to see everything?
  • Ritual's drama — should the lock always feel ceremonial?

Brand Voice

  • Where should New York serif appear — everywhere or only voice?
  • How literal should "past self / future self" language be?
  • Should the tone be sacred/minimal (Oracle) or warm/functional (Architecture)?
  • Does the commitment lock need its own visual register?

Hybrid Options

  • Oracle's Now tab + Architecture's Today tab structure
  • Editorial's typography + Signal's health widget
  • Ritual's lock ceremony + Oracle's completion moment
  • Deep Clay dark mode + any light direction as base
